# HelloWorld

Un semplice sketch **“Hello World”** per iniziare a visualizzare informazioni sullo schermo del modulo **ESP32-4848S040**.
Serve come base per comprendere la gestione del display e sviluppare applicazioni più complesse.

## Descrizione

Il programma mostra un messaggio di benvenuto sullo schermo, utile per testare il corretto funzionamento del display, della libreria grafica e della comunicazione con la scheda.

## Scopo

Questo progetto è pensato come punto di partenza per chi desidera:

* Verificare la corretta connessione hardware del modulo;
* Familiarizzare con le funzioni grafiche di base;
* Costruire interfacce più evolute a partire da un esempio minimale.

## English Version

# HelloWorld

A simple **“Hello World”** sketch to start displaying information on the **ESP32-4848S040** module screen. It serves as a base
line for understanding how to manage the display and build more complex applications.

## Description

The program shows a welcome message on the screen, useful for testing the display, the graphics library, and communication with
the board.

## Purpose

This project is intended as a starting point for those who want to:

* Check that the hardware connections of the module are correct;
* Get familiar with the basic graphics functions;
* Build richer interfaces beginning from a minimal example.
