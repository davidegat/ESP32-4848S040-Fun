# HelloWorld

Un semplice sketch **“Hello World”** per iniziare a visualizzare informazioni sullo schermo del modulo **ESP32-4848S040**.
Serve come base per comprendere la gestione del display e sviluppare applicazioni più complesse.

## Descrizione

Il programma mostra un messaggio di benvenuto sullo schermo, utile per testare il corretto funzionamento del display, della libreria grafica e della comunicazione con la scheda.

## Scopo

Questo progetto è pensato come punto di partenza per chi desidera:

* Verificare la corretta connessione hardware del modulo;
* Familiarizzare con le funzioni grafiche di base;
* Costruire interfacce più evolute a partire da un esempio minimale.
